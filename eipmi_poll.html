Behaviours: gen_server.

Description

A server that periodically polls information for a specific IPMI target. The server monitors its corresponding eipmi_session process and exits as soon as the session gets down.

Currently only polling of the target's System Event Log (SEL) is supported. All entries retrieved from the SEL will be forwarded as asynchronous events to the currently subscribed handlers using eipmi_events.
