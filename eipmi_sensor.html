Function Details

get_addr/1

get_addr(X1::binary()) -> [addr()]

Decodes the sensor from a 2 or 3 byte binary according to the encoding provided by SEL and SDR records.

get_entity/2

get_entity(Id::non_neg_integer(), Instance::binary() | non_neg_integer()) -> [entity()]

Returns a list containing the decoded (human readable) entity properties. To retrieve only the entity id use get_entity_id/1.

get_entity_id/1

get_entity_id(Id::non_neg_integer()) -> [{entity_id, entity_id()}]

Returns the human readable entity for a given entity id according to section 43.14 from the IPMI specification. For full entity decoding refer to get_entity/2.

get_type/2

get_type(ReadingType::non_neg_integer(), SensorType::non_neg_integer()) -> {threshold | discrete | {oem, non_neg_integer()}, type()}

Returns the sensor category and reading type from the given sensor and reading type. See sections 41 and 42 of the IPMI specification for more information.

get_unit/1

get_unit(X1::non_neg_integer()) -> unit()

Returns the sensor unit from the given type code according to section 43.17 from the IPMI specification.

get_value/5

get_value(Type::type(), Offset::non_neg_integer(), Assertion::0 | 1, Value1::0..255, Value2::0..255) -> [value()]

Returns a (raw) sensor reading. The returned reading may have to be converted into human readable values (at least for threshold based sensors).
