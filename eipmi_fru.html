Function Details

read/2

read(SessionPid::pid(), FruId::0..254) -> {ok, info()} | {error, term()}

Read and decode FRU inventory data for a specific FRU id using the provided IPMI session. The returned FRU information is a property list that does only contain the available and checksum error free fields of the inventory. If no FRU data is available for a specific id the returned inventory data is the empty list.
