Function Details

clear/1

clear(SessionPid::pid()) -> ok | {error, term()}

Clear the system event log regardless whether events have been read/processed by any user application.

read/2

read(SessionPid::pid(), X2::boolean()) -> [entry()]

Read all entries from the system event log and optionally clear the log. Reading the SEL without clearing it is implemented in a way that will only send the least number of packets (no SEL reservation or info requests). Thus, if a user knows that read SEL entries will be cleared automatically by the BMC the user should not clear the SEL explicitly.
